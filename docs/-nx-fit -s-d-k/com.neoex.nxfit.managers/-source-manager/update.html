update

abstract suspend fun update(vararg sources: SourceModel)

Update sources by replacing them based on ID. If the ID fails to find a match, it will be ignored. This can not be used to create new sources.

Any call to update() will trigger a sync processes. This will ensure that the server receives the updates as well. At this time, if the local list of sources is out of date with the server, it will be updated.

Will publish the updated list of sources via the sources FlowState.

Parameters

sources

One or more sources to be updated.
