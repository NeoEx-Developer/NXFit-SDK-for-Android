PublicSessionClient

interface PublicSessionClient

Interface to the Public Session resource. This provides a means to view the public properties of a session for any user, i.e. not the logged in user. Each of these calls require a user ID to be provided. It also provides a means to list publicly available sessions sessions. These can be filtered by date range and Studio.
