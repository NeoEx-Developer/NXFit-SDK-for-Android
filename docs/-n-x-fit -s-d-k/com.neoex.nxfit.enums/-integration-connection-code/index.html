IntegrationConnectionCode

enum IntegrationConnectionCode : Enum<IntegrationConnectionCode>

When the Garmin Connect consent process completes, a result is provided via an Intent sent back to the app. The Intent can be decoded with the IntegrationsManager.handleAuthorizeResponse method, which will return the result in the form of this enum.
