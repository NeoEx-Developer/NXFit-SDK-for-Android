build

fun build(appContext: Context, configProvider: ConfigurationProvider, userId: Int, accessTokenFlow: StateFlow<String?>): NXFit

Builds a concrete implementation of the NXFit interface. When the build completes, all the contained clients and managers are fully initialized and ready to use.

Every call to this returns a new instance of NxFit. It's recommended that a single instance is used for a given user ID. When a user logs out or the instance is no longer required, close should be called to free up resources.

Return

A new instance of NxFit.

Parameters

appContext

Should be an Application Context

configProvider

Used to provide the baseUrl along with other configurable features.

userId

ID for the current user. A userId is always required, even if there isn't an access token available.

accessTokenFlow

A flow to access tokens that are used when making HTTP calls to the NxFit API server.
