Creates a new session.

Parameters

externalId

The unique identifier of the activity. This is typically created by the session source (eg Health Connect). This will not be used as the session ID, but rather as a reference to the original activity from the source.

activityType

The type of activity being performed (e.g., running, cycling).

integrationIdentifier

The identifier for the integration that created this session.

sourceDeviceName

The name of the device that recorded the session.

sourceDeviceHardwareVersion

The hardware version of the source device.

sourceDeviceManufacturer

The manufacturer of the source device.

sourceDeviceOS

The operating system of the source device.

sourceDeviceOSVersion

The OS version of the source device.

sourceAppName

The name of the application that recorded the session.

sourceAppIdentifier

The identifier for the application that recorded the session.

syncDeviceName

The name of the device used for synchronization, if applicable.

syncDeviceManufacturer

The manufacturer of the synchronization device, if applicable.

syncDeviceOS

The operating system of the synchronization device, if applicable.

syncDeviceOSVersion

The OS version of the synchronization device, if applicable.

syncDeviceAppVersion

The app version on the synchronization device, if applicable.

syncId

An identifier for synchronization, if applicable.

syncVersion

The version of synchronization, if applicable.

startedOnLocal

When this session started on the local device.

endedOnLocal

When this session ended on the local device.

activeDurationInSeconds

Duration in seconds that this session was active.
