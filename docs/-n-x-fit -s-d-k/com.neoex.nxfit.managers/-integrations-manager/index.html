IntegrationsManager

interface IntegrationsManager : AutoCloseable

Provides methods to manage the user's integrations fitness providers. Connecting an user to an integration means that fitness data from that integration will be synced and made available for this user.

NOTE: The manager maintains state and is designed to be used as a singleton.
