CadenceMetrics

data class CadenceMetrics(val avgCadencePerMinute: Double, val maxCadencePerMinute: Double, val minCadencePerMinute: Double, val cadenceUnitShort: String?, val cadenceUnitFull: String?)

Represents detailed cadence metrics for a fitness session.

Cadence is the rate at which a person performs a repetitive motion, such as steps per minute in running or revolutions per minute (RPM) in cycling. This class captures the average, maximum, and minimum cadence values recorded during the session, along with the units of measurement.
