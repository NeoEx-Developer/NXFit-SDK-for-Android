IntegrationModel

data class IntegrationModel(val identifier: String, val displayName: String, val logoUrl: String, val isConnected: Boolean, val isLocal: Boolean, val updatedOn: ZonedDateTime)

Represents a single integration partner or data source within the application.

This data class holds all the necessary information to display and manage an integration, such as a connection to a fitness tracker service or a local health data source.
