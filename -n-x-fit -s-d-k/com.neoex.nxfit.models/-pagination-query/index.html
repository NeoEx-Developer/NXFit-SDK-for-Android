interface PaginationQuery

Represents a query for paginated data, allowing for fetching pages of results.

This sealed interface defines the common properties for pagination: limit, afterToken, and beforeToken. It provides different implementations to handle various pagination scenarios, such as fetching the first page, the next page, the previous page, or a specific page defined by either token.
