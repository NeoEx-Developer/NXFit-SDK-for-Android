Query

data class Query(val afterToken: String?, val beforeToken: String?, val limit: UInt = defaultLimit) : PaginationQuery

Represents a generic pagination query that can specify either an afterToken, a beforeToken, or neither, along with a page limit.

This class provides flexibility for creating pagination requests but includes a check to ensure that afterToken and beforeToken are not used simultaneously, as that would be an invalid state. For more specific use cases, prefer using LimitQuery, AfterQuery, or BeforeQuery.

Throws

Exception

if both afterToken and beforeToken are provided with non-empty values.
