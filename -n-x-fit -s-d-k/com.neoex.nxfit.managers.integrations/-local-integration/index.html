LocalIntegration

interface LocalIntegration

Defines the contract for a local integration. Local integrations are those where the health and fitness data resides on the same device as the application, rather than in the cloud. Examples include Google Health Connect on Android.

Implement this interface to create a custom local integration and register it with the IntegrationsManager using IntegrationsManager.registerLocalIntegration. You can think of this as a plugin for the IntegrationsManager. The IntegrationsManager.connect and IntegrationsManager.disconnect methods are essentially delegated by the IntegrationsManager to the local integration.

The NXFitHealthConnect integration is an example of a local integration. It will automatically register itself when you create an instance via the NXFitHealthConnect.build factory.
