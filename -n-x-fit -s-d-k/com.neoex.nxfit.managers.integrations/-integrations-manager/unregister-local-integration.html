unregisterLocalIntegration

abstract fun unregisterLocalIntegration(integrationIdentifier: String)

Unregisters a local integration.

Once a local integration is unregistered, it will be removed from the list emitted by the integrations flow and will no longer be available for connection. If the integration was previously connected, it will remain connected even though the integration is no longer listed in integrations.

The expectation is that this only be called when shutting down the NXFit SDK and this allows a mechanism to ensure no memory leaks occur.

Parameters

integrationIdentifier

The identifier of the local integration to unregister. This should match the LocalIntegration.identifier that was used for registration.
