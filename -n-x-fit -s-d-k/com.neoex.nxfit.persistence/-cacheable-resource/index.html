CacheableResource

data class CacheableResource<T>(val value: T, val lastModifiedOn: ZonedDateTime? = null, val eTag: String? = null) : Cacheable, Resource<T>

CacheableResponse represents a network resource that may be cacheable. Either an eTag and/or the last modified date may be used for change detection. Future requests for that resource may include either the eTag or last modified date. If the resource has not changed based on that criteria a 304 NOT MODIFIED status code will be returned with no content. The Client methods that return a CacheableResource will throw a NotModifiedException whenever that happens.

The eTag as calculated by the server. The eTag is a hash based on the value of the resource. This means changes to the value result in changes to the eTag. The client should never attempt to calculate the eTag but instead to cache the value supplied by the server and use it in subsequent requests.
