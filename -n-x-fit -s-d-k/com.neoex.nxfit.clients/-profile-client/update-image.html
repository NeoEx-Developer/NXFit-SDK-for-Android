updateImage

abstract suspend fun updateImage(filePath: String): Uri

Given the path to an image file, will upload the image to the server and make it the current profile image for the current user.

Before the image is sent it will be converted to PNG

Return

Returns the Uri of the new profile image, which will be different every time the profile image is updated.

Parameters

filePath

The path to a file. This would be equivalent to what you'd pass to the File() constructor or retrieve from File.path.

Throws

ImageFileNotFoundException
UnsupportedImageFileException
