NXFit

interface NXFit : AutoCloseable

The NXFit interface provides all the clients and managers needed to access NXFit services. Use the NXFit.build method to build an instance of this.

Both Clients and Managers are used to access data from the NXFit API service, however, managers provide extra functionality in the form of state management. Managers use local storage to cache data and make reads and writes possible even without network connectivity. Managers may also provide a notification mechanism to indicate changes in data.

NOTE: The IntegrationsManager provides a mechanism to connect and disconnect integrations for the current user. See IntegrationsManager.connect for more details.

abstract val accessTokenFlow: StateFlow<String?>

A flow that provides access tokens used when making HTTP calls to the NxFit API server. The flow may emit null to indicate that there is no access token available. In this case, only endpoints that do not require authentication will succeed. This flow matches the one provided when building this instance and is associated with the user ID.
