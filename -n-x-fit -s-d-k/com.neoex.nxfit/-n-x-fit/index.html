interface NXFit

The NXFit interface provides all the clients and managers needed to access NXFit services. Use the NXFit.build method to build an instance of this.

Both Clients and Managers are used to access data from the NXFit API service, however, managers provide extra functionality in the form of state management. Managers use local storage to cache data and make reads and writes possible even without network connectivity. Managers may also provide a notification mechanism to indicate changes in data.

NOTE: The IntegrationsManager provides a mechanism to connect and disconnect integrations for the current user. See IntegrationsManager.getAuthorizeUri for more details.
